    • Defined in packages/core/src/RenderPipeline/BasicRenderPipeline.ts:61

    Add render pass.

    Parameters

    • nameOrPass: string | RenderPass

      The name of this Pass or RenderPass object. When it is a name, the following parameters need to be provided

    • Default value priority: number = null

      Priority, less than 0 before the default pass, greater than 0 after the default pass

    • Default value renderTarget: RenderTarget = null

      The specified Render Target

    • Default value replaceMaterial: Material = null

      Replaced material

    • Default value mask: Layer = null

      Perform bit and operations with Entity.Layer to filter the objects that this Pass needs to render

    • Default value clearParam: Vector4 = new Vector4(0, 0, 0, 0)

      Clear the background color of renderTarget
